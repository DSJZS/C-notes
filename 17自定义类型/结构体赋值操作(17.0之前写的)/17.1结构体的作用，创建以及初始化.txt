*结构体的作用
之前学习到的类型如 int ，char等可以描述单个数据
如年龄，名字首字母等，但是无法描述一个人的所有信息
如果我们想描述所有信息，即描述一个复杂对象
就需要结构体

*结构体的创建
结构体是需要我们定义的一种复杂变量
其内部可含多个变量
如我们要储存一本书的信息(如书名，书价)
struct BOOK
{
	char name[20] // 用于储存书名的字符数组
	short price； // 用于储存价格的短整型
}; // 注意 ; 号

*结构体的初始化
如上面的结构体类型
其初始化方式为按照顺序填入其所需内容
struct BOOK book = { "C语言程序设计", 55 };
