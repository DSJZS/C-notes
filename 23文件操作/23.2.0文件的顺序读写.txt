功能			函数名			适用于
字符输入函数		fgetc			所有输入流	( 文件，标准输入流-键盘 )
字符输出函数		fputc			所有输出流	( 文件，标准输出流-屏幕 )
文本行输入函数		fgets			所有输入流	( 文件，标准输入流-键盘 )
文本行输出函数		fputs			所有输出流	( 文件，标准输出流-屏幕 )
格式化输入函数		fscanf			所有输入流	( 文件，标准输入流-键盘 )
格式化输出函数		fprintf			所有输出流	( 文件，标准输出流-屏幕 )
二进制/文本输入		fread			文件
二进制/文本输出		fwrite			文件

ps:
标准输入流：stdin	对应键盘
标准输出流：stdout	对应屏幕

对文件而言：
fread / fwrite 函数 既可以操作 二进制文件 , 又可以操作 文本文件 ;
getc / putc 函数 , fscanf / fprintf 函数 , fgets / fgets 函数 , 只能用于操作 文本文件 ;
字符一律以ASCII形式存储，数值型数据既可以用ASCII形式存储，也可以使用二进制形式存储。
