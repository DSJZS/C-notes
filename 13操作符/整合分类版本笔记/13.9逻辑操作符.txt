逻辑操作符：
&&	逻辑与
||	逻辑或

&& 逻辑与：
左右两边均为真则为真，否则为假
可用作“并且”

|| 逻辑或：
左右两边至少有一个为真则为真，否则为假
可用作“或者”

*注意：&& 一旦左边为假，便不会在执行右边的表达式
如	int i=0,a=0,b = 2,c=3,d=4;
	i=a++ && ++b && d++;
	printf("%d%d%d%d",a,b,c,d);
	// 输出 1 2 3 4

*注意：|| 一旦左边为真，便不会在执行右边的表达式
如	int i = 0, a = 0, b = 2, c = 3, d = 4;
	i = a++ || ++b || d++;
	printf("%d%d%d%d",a, b, c, d);
	// 输出 1 3 3 4

*注意：不要和 &按位与 还有 |按位或 弄混
