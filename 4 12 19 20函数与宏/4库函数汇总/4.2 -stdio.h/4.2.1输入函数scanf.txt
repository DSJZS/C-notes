用于在控制台上输入数据
头文件: stdio.h

调用格式:
scanf("<格式化字符串>", <参量表>);
格式化字符串即为要输入的内容，其中可输入表示
一些数据的规定符，而参量表中要填写这些数据的地址
输入时数值并回车后这些数据会被依次赋值

*注意1
scanf会从输入缓冲区中读取数据，空格会分割输入的数据，比如输入一个包含空格的字符串时用scanf读取字符串，它只会读取空格前的内容

*注意2
如果"<格式化字符串>"中除了规定符还填写了其它内容，输入时要加上
如	int a = 0;
	int b = 0;
	scanf("%d&%d", &a, &b); // 输入 88&6 
	printf("%d %d", a, b); // 输出 88 6

举例 
    int a = 0;
    int b = 0;
    scanf("%d%d", &a, &b); // 输入 88 6 7
    printf("%d %d", a, b); // 输出 88 6



规定符
%d 十进制有符号整数
%u 十进制无符号整数
%f 浮点数
%s 字符串
%c 单个字符
%p 指针的值
%e 指数形式的浮点数
%x, %X 无符号以十六进制表示的整数
%o 无符号以八进制表示的整数
%g 把输出的值按照 %e 或者 %f 类型中输出长度较小的方式输出
%p 输出地址符
%lu 32位无符号整数
%llu 64位无符号整数
%% 输出百分号字符本身。
除了格式化说明符之外，printf() 函数还支持一些标志和选项，用于控制输出的精度、宽度、填充字符和对齐方式等。例如：
%-10s：左对齐并占用宽度为 10 的字符串；
%5.2f：右对齐并占用宽度为 5，保留两位小数的浮点数；
%#x：输出带有 0x 前缀的十六进制数。
