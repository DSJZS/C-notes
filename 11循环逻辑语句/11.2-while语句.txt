while语句用于执行循环逻辑

语法：
while(表达式)
    循环语句;

进入while语句时，会判断表达式是否为真
如果为真，则执行代码块里的语句
代码块内语句执行完后会判断表达式是否还为真
如果还为真则再执行代码块内语句，后再判断表达式
是否还为真以此循环。
一旦判断的时候表达式为假，则不再循环

如
	int i = 1;
	while (i <= 5)
	{
		printf("%d", i);
		i++;
	}
	// 输出 12345
*关键字 break
while等循环语句执行时遇到 break语句 会立刻跳出循环不再执行任何循环语句
-用于永久终止循环
如
	int i = 1;
	while (i <= 5)
	{
		if (i == 3)
			break;
		printf("%d", i);
		i++;
	}
	// 输出 12


*关键字 continue
while等循环语句执行时遇到 continue语句 会立刻跳出当前的这一次循环开始下一次循环(即不再继续执行本次循环的任何语句，直接开始判断表达式是否为真)-
-用于终止本次循环，进入下一次循环的判断入口
如
	int i = 1;
	while (i <= 5)
	{
		if (i == 3)
			continue;
		printf("%d", i);
		i++;
	}
	// 输出 12
	// 然后死循环
