字符指针即 char* 类型的指针
字符数组的数组名存放着首元素的地址
因此可以用字符指针存放首元素的地址，类似数组名(只是类似)

基于以上内容(只是打个比方)
我们可以一个字符串常量 "abc" (注意"abc"是一个常量,其地址是固定的,而且内容理论上不可被修改,因此后面要用常量指针)的首地址存入常量字符指针
const char* ps = "abc";  

*注意：字符数组存放的是字符串变量,字符串元素可变
      而常量字符指针存放的是字符串常量,字符串元素不可变

*注意：由于上文 "abc" 是一个常量,在内存中为节省空间只会创建一份 "abc"
如	char* p1 = "abc";
	char* p2 = "abc";
	if (p1 == p2)
		printf("p1=p2");
	// 输出 p1=p2
	// 说明 p1和p2指向的是一块空间,"abc"只被创建一次

*注意：字符指针存放字符串常量首元素地址时用关键字sizeof检测其大小时，不会和数组名一样代表整个字符串，它只不过是个指针( 大小为 4/8 )
